# 🎌 Nihongo Master

**Nihongo Master** is an interactive, gamified web application designed to help users learn Japanese effectively and stay motivated. 

Built with React and Vite, the application features daily learning streaks, quiz questions to test your knowledge, an engaging user interface, and profile persistence to track your progress over time.

---

## ✨ Features

- **🏆 Gamified Learning**: Stay motivated with daily learning streaks that track your consistency.
- **📝 Daily Questions**: Test your Japanese knowledge with fresh questions every day. 
- **💾 Profile Persistence**: Your progress, including profile picture, username, leaderboard scores, and streak data, is saved securely so you never lose your progress.
- **⚡ Fast Performance**: Built using modern web development tools like Vite and React for a highly responsive experience.
- **🎨 Beautiful UI**: A clean, modern interface designed specifically to make language learning enjoyable rather than a chore.

---

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Styling**: Vanilla CSS / Tailwind CSS (Optional based on implementation)
- **Deployment**: Netlify ready

## 🚀 Running the App Locally

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/HarshitPanchal04/Nihongo-master-app.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Nihongo-master-app
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser** and navigate to the localhost URL provided in your terminal (typically `http://localhost:5173`).

## ⚙️ Deployment

This application is configured for seamless deployment on **Netlify**. 
Simply connect the repository to your Netlify account, set the build command to `npm run build`, and the publish directory to `dist`.
